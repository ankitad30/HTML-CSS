SORTING ALGORITHMS

We are going to discuss some common Sorting algorithms.

A sorting Algorithm is an algorithm that arranges the elements of a list in a certain order.Sorting is used to optimize the use of other algorithms for eg. Search and Merge algos which require input data to be in sorted lists; it is also often useful for canonicalizing data and for producing human-readable output.

  1. Computational Complexity
  2. Memory Usage
  3. Recursion
  4. Stability
  5. In-Place or Out-of-Place
